# FruitFreshnessCNN

## Overview
FruitFreshnessCNN is a deep learning-based image classification project designed to identify fresh and rotten fruits using convolutional neural networks (CNNs).

The project classifies fruit images into six categories:
- Fresh Apples
- Fresh Bananas
- Fresh Oranges
- Rotten Apples
- Rotten Bananas
- Rotten Oranges

The system was developed using TensorFlow and Keras and includes data preprocessing, image augmentation, model training, evaluation, and prediction visualization.

---

## Objectives
- Build a CNN model for fruit freshness classification
- Detect fresh and rotten fruits from images
- Improve classification performance using image augmentation
- Evaluate model accuracy and prediction performance
- Visualize predictions and training results
